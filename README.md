# Attendance Tracker Setup Script

# What it does
Creates an attendance tracking project with proper folder structure.

# How to use
1. Run: `./setup.sh`
2. Enter a project name
3. Choose if you want to change threshold values or keep the defaults
4. Done!

# Running the tracker
`cd attendance_tracker_{name} && python3 attendance_checker.py`


# Testing the archive feature
Press Ctrl+C while the script is running. It will create a backup and clean up.
